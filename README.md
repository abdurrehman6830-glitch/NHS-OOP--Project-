# NHS-OOP--Project-
Object Oriented Programming based NHS Appointment Management System with MySQL JDBC
NHS Appointment Management System
Object-Oriented Programming Project
This project presents a digital solution to reduce NHS waiting times and missed appointments using OOP principles in Java, MySQL database with JDBC connectivity, and responsive GUI.
Features:
Online appointment booking, rescheduling & cancellation
Automated SMS/Email reminders
Video consultation option
Patient and Admin dashboards
Technical Stack:
Backend: Java (OOP)
Database: MySQL with JDBC
Frontend: HTML5, CSS3, JavaScript
UML Diagrams: Visual Paradigm
Class Diagrams and Use Case Diagrams are included in the repository.
